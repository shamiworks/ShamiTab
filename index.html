Getting started

Hover over the notation area to highlight time divisions and string slots. Click a string slot to select it — selected slots are shown with an outline. You can then enter a tsubo using the keyboard or palette. Click a time division outside a string slot to select the division only — this is needed for some techniques.

Changing the number of measures

Click anywhere on a staff unit to select it — the unit is outlined when selected. Then click Free, 2, 4, or 8 in the palette.

Changing barline types

Alt+click a barline to select it, then choose normal, repeat, or stop from the palette.

Suri and oshibachi

Enter the starting tsubo and the target tsubo. Select the starting tsubo slot. Press I for suri or O for oshibachi. The arc will be drawn between the two tsubo. Both must be in the same staff unit.

Triplets

Select a time division and press T. Three consecutive divisions are grouped as a triplet and a bracket is drawn below the staff. The fourth division is disabled. To remove a triplet, select any active tsubo in the group and press T again, or press Backspace.
